Решение 1:

```sh
cat ex2.txt | tr -s ' ' | cut -d ' ' -f 4- | awk '{print NR ".", $0}' | sort -t " " -k 2
```

```sh
cat ex2.txt | awk '{$1=$2=$3=""; print}' | awk '$1=$1' | awk '{print NR ".", $0}' | sort -t " " -k 2
```
